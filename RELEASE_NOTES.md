# v0.1 测试版

这是第一个测试版。

## 怎么使用

1. 下载 `pcr-rogue-helper-v0.1-windows-app.zip`。
2. 解压压缩包。
3. 打开雷电模拟器 9，并进入游戏。
4. 建议模拟器分辨率设置为 `1280x720`。
5. 双击 `PCRRogueHelper.exe` 开始运行。

## 说明

- 找到符合条件的 Boss 组合后，工具会自动停止。
- 默认有效组合已经内置，不需要额外设置。
- 如果想自己选择有效组合，可以用程序的配置界面生成 `valid_combos.json`。
- 这是测试版。如果识别不准，可以替换压缩包里的截图和 Boss 图标。
