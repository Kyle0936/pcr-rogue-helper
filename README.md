# PCR Rogue Helper

这是一个用于雷电模拟器 9 的 PCR Rogue 开局筛选小工具。

工具会按照 `1.png` 到 `9.png` 的截图流程自动点击，并检查第三层 Boss 和第五层 Boss 是否符合设定组合。找到有效组合后，工具会关闭 Boss 弹窗并停止，不会继续点击截图 8 的按钮。

## 普通用户快速开始

1. 在发布页面下载 `pcr-rogue-helper-v0.2-windows-app.zip`。
2. 解压到任意文件夹。
3. 打开雷电模拟器 9，并进入游戏。
4. 建议模拟器分辨率保持为 `1280x720`，运行过程中不要改分辨率。
5. 双击 `PCRRogueHelper.exe`。
6. 在中文界面里选择有效 Boss 组合，然后点击“保存并开始”。
7. 同一个页面上方可以调整成功条件和控制程序，下方会显示运行进度、错误提示和成功结果。

发布包已经包含可直接运行的 Windows 程序，不需要安装 Python。

## 默认有效组合

- 五王 `boss51` + 三王 `boss31` / `boss32` / `boss33`
- 五王 `boss53` + 三王 `boss32` / `boss33`

`boss34`、`boss35`、`boss54`、`boss55` 也会被识别，但默认都算作无效组合。

## 自定义有效组合

双击 `PCRRogueHelper.exe` 后会自动打开中文界面。界面里每一行是一组有效组合，可以用下拉框选择五王和三王，旁边会显示对应图标。

- 点击“添加组合”可以增加一组有效组合。
- 点击“删除”可以移除一组组合。
- 点击“恢复默认”会恢复内置默认组合。
- 点击“保存并开始”会保存组合并开始运行。
- 点击“保存组合”只保存，不开始运行。

## 控制和运行状态

工具页面分为上下两部分：

- 上半部分：设置成功条件，并控制程序开始、暂停、继续、结束。
- 下半部分：显示运行状态、识别进度、错误提示和成功结果。

运行中可以使用这些按钮：

- “暂停”：临时停止自动流程，方便修改有效组合。
- “继续”：保存当前组合，并用新的条件继续运行。
- “结束”：结束当前任务，但保留页面，可以修改条件后重新开始。
- “保存组合”：只保存条件，不启动或继续。
- “关闭工具”：结束后台程序并关闭控制页面。

直接关闭浏览器页面时，工具也会自动结束后台程序。关闭后再删除解压出来的文件夹，就不会再出现文件夹被占用的问题。

状态区域会显示：

- 正在连接雷电模拟器。
- 正在检查当前流程画面。
- 正在识别五王和三王。
- 当前组合无效时，会提示继续下一轮。
- 找到目标组合时，会显示成功消息。
- 如果找不到雷电模拟器或 ADB 无法连接，会显示中文错误提示。

## 使用条件

- 雷电模拟器 9 需要保持运行。
- 游戏需要已经打开到可操作画面。
- 默认通过 ADB 与模拟器交互，模拟器窗口不一定需要在最前面。
- 如果模拟器设备编号不是 `emulator-5554`，需要通过命令行参数指定。
- 运行过程中不要改变模拟器分辨率。

## 常见问题

### 为什么只提供一个压缩包？

发布页面只需要下载 `pcr-rogue-helper-v0.2-windows-app.zip`。这个包里包含 `PCRRogueHelper.exe` 和 `screenshots` 图片文件夹，是给普通用户直接使用的版本。

### 识别不准怎么办？

重新截取对应步骤的图片，并替换 `screenshots` 文件夹里的同名文件，例如 `5.png`、`detect3.png`、`detect5.png`。红色方框或红圈只是手动标记，工具在识别画面时会自动忽略这些红色标记。

### 找到目标组合后会发生什么？

工具会关闭 Boss 信息弹窗，然后在页面下方显示成功。它不会继续点击截图 8 的按钮。

### 没找到目标组合怎么办？

工具会关闭弹窗，继续下一轮刷新流程。

## 源码运行

如果你想从源码运行，而不是使用发布包里的 `.exe`：

```powershell
python -m pip install -r requirements.txt
python pcr_rogue_helper.py
```
