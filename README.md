# PCR Rogue Helper

这是一个用于雷电模拟器 9 的 PCR Rogue 开局筛选小工具。

脚本会按照 `1.png` 到 `9.png` 的截图流程自动点击，并检查第三层 Boss 和第五层 Boss 是否符合你选择的组合。找到符合条件的组合后，脚本会自动停止，不会继续点击下一步。

## 快速开始

1. 下载发布页里的压缩包，并解压到任意文件夹。
2. 打开雷电模拟器 9，并进入游戏。
3. 建议模拟器分辨率保持为 `1280x720`。
4. 第一次使用时，双击 `install_dependencies.bat`。
5. 使用默认规则时，双击 `run_helper.bat`。

默认有效组合：

- 五王 `boss51` + 三王 `boss31` / `boss32` / `boss33`
- 五王 `boss53` + 三王 `boss32` / `boss33`

`boss34`、`boss35`、`boss54`、`boss55` 默认会识别，但会被当作无效组合。

## 自定义有效组合

如果想自己选择哪些 Boss 组合算有效：

1. 双击 `configure_combos.bat`。
2. 在弹出的中文界面里，根据图标勾选有效组合。
3. 点击 `保存并退出`。
4. 之后双击 `run_custom_combos.bat`。

配置会保存到 `valid_combos.json`。想恢复默认规则，可以重新打开配置界面并点击 `恢复默认`。

## 使用条件

- 雷电模拟器 9 需要保持运行。
- 游戏需要已经打开。
- 默认使用 ADB 与模拟器交互，雷电窗口不一定需要在最前面。
- 如果模拟器设备编号不是 `emulator-5554`，需要用命令行参数指定。
- 不要在脚本运行时改变模拟器分辨率。

## 常见问题

### 脚本识别不准怎么办？

重新截取对应步骤的图片，并替换同名文件，例如 `5.png`、`detect3.png`、`detect5.png`。

红色方框或红圈只是手动标记，脚本在识别画面时会自动忽略这些红色标记。

### 找到目标组合后会发生什么？

脚本会关闭 Boss 信息弹窗，然后直接停止。它不会继续点击截图 8 的按钮。

### 没找到目标组合怎么办？

脚本会关闭弹窗，继续下一轮刷新流程。

## 文件说明

- `run_helper.bat`：使用默认规则运行。
- `configure_combos.bat`：打开组合选择界面。
- `run_custom_combos.bat`：使用自定义组合运行。
- `install_dependencies.bat`：第一次使用时安装依赖。
- `1.png` 到 `9.png`：流程截图。
- `detect3.png`、`detect5.png`：Boss 检查截图。
- `boss*.png`：Boss 图标。
